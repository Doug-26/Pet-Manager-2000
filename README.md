# Pet Manager 2000

## Description
Pet Manager 2000 is a full-stack application built with Angular and .NET for managing pet records, appointments, and owner information.

## Tech Stack
- **Frontend:** Angular
- **Backend:** .NET
- **Database:** SQL Server (or your database choice)

## Features
- Pet profile management
- Owner information tracking
- Appointment scheduling
- Health records storage

## Getting Started

### Prerequisites
- Node.js and npm
- .NET SDK
- Visual Studio Code or Visual Studio

### Installation
1. Clone the repository
2. Install frontend dependencies: `npm install`
3. Build and run the .NET backend
4. Start the Angular development server: `ng serve`

## Project Structure
```
Pet Manager 2000/
├── frontend/          (Angular application)
└── backend/           (.NET API)
```
